# [Guess The Number]

A small JavaScript game where the player tries to guess a random number between **1 and 20**.

## Features

- Generates a random number each round
- Displays messages based on the guess
- Shows the number of attempts (score)
- Saves the best result using **Local Storage**
- Includes a restart button

## How to Play

Enter a number and click **Check**.  
Try to guess the correct number with as few attempts as possible!

## Technologies

HTML • CSS • JavaScript
